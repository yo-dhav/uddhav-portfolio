# Uddhav Krishna - Professional Portfolio

**[Live Website: https://yo-dhav.github.io/uddhav-portfolio/](https://yo-dhav.github.io/uddhav-portfolio/)**

Welcome to my personal portfolio website! This project is a modern, visually striking showcase of my skills, projects, and learning journey as a Computer Science student specializing in Data Science and Machine Learning.

## ✨ Features
- **Dark Mode & Glassmorphism Aesthetics:** Sleek, high-contrast futuristic design using soft transparent UI cards and neon glowing accents.
- **Interactive Magnetic Cursor:** Custom cursor built from scratch that smoothly trails your mouse movements and transforms dynamically over interactive elements.
- **3D Tilt Elements:** Integrated `Vanilla-Tilt.js` for an engaging 3D parallax hover effect across project cards.
- **Responsive Layout:** fully optimized for both desktop and mobile viewing using robust CSS Grid and Flexbox logic.

## 🚀 How to Host This Site for Free

You can make this website live on the internet using GitHub Pages! It's completely free and takes less than a minute.

1. Go to this repository's settings page: [Settings > Pages](https://github.com/yo-dhav/uddhav-portfolio/settings/pages)
2. Look for the **"Build and deployment"** section.
3. Change the **"Source"** dropdown to **"Deploy from a branch"**.
4. In the **"Branch"** dropdown right below it, select **`main`** and click **Save**.

Wait about 60 seconds, and GitHub will publish your site! A live link will appear at the top of the settings page (e.g. `https://yo-dhav.github.io/uddhav-portfolio/`) which you can share on your resume and LinkedIn!
